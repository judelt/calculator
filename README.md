### Setup

1. Fork and clone this repository.<br />
2. Install dependencies with `npm install`.<br />
3. Then run `npm run serve`.
4. Go to `http://localhost:8080/` in your browser.

### Dependencies

- Vue v.2.6.11
- Core-js v.3.6.5
